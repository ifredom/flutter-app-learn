# first_flutter_app

A new Flutter application. flutter_learn_ex中是具体的学习步骤以及每一个组件的练习代码



## 开发必备

```bash
  # 运行.项目根目录下
  > flutter run
    # 运行.项目根目录下
  > flutter run -d [设备名称]

```

## debug

* 按下R 刷新
* 按下P 出现网格
* 按下O 切换android和ios视图

## windows 系统必知快捷键

alt + click（鼠标左键）  进入该对象查看属性
alt + left (左箭头)  后退返回的项目页
